About AIBugHunter

Line-level Vulnerability Predictions

Given a file written C/C++, AIBugHunter scans the file and predict the potentially vulnerable lines existing in each function.

A predicted vulnerable line is highlighted by a red underline.

Vulnerability Type & Severity Predictions

If a function is detected as a vulnerable function. AIBugHunter shows the information of CWE-ID, Abstract CWE Type, and the CVSS Severity Score.

Vulnerability information can be obtained by hovering over the vulnerable line and clicking the "view problem" button.
